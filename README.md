# Housing Price Prediction

This project predicts house prices using Machine Learning based on property features such as area, bedrooms, bathrooms, stories, parking, and furnishing status.

## Project Overview
- Data loading and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization
- Feature engineering
- Model training and evaluation
- House price prediction

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Dataset
The dataset contains housing information, including:
- Price
- Area
- Bedrooms
- Bathrooms
- Stories
- Main Road
- Guest Room
- Basement
- Hot Water Heating
- Air Conditioning
- Parking
- Preferred Area
- Furnishing Status

## Result
The model predicts house prices based on various housing features and helps analyze the factors affecting property prices.

## How to Run
1. Open the Jupyter Notebook.
2. Install the required Python libraries.
3. Load the dataset.
4. Run all notebook cells.
5. View the model predictions and results.
